# ⚡ `nhash id cuid` - Secure CUID2 Generator

Generates highly secure, next-generation CUID2 identifiers (collision-resistant).

---

## 🛠️ Usage Syntax

```bash
nhash id cuid [arguments] [options]
```

### 💡 Live Terminal Examples

**Generate a secure CUID2 ID**:
```bash
nhash id cuid
```


---

> [!TIP]
> All subcommands in **nHash** inherit the global `--output` (`-o`) redirection parameter to save execution results directly to a local file.
> ```bash
> nhash id cuid --output result.txt
> ```
