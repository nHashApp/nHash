# ⚡ `nhash text lorem` - Lorem Ipsum Paragraphs Generator

Generates paragraphs of Lorem Ipsum placeholder text.

---

## 🛠️ Usage Syntax

```bash
nhash text lorem [arguments] [options]
```

### ⚙️ Options & Parameters

| Option Flag | Short Flag | Type | Description | Default Value |
| :--- | :--- | :--- | :--- | :--- |
| `--paragraphs` | `-p` | `int` | Count of paragraphs to generate. | `1` |

### 💡 Live Terminal Examples

**Generate 3 paragraphs of placeholder text**:
```bash
nhash text lorem -p 3
```


---

> [!TIP]
> All subcommands in **nHash** inherit the global `--output` (`-o`) redirection parameter to save execution results directly to a local file.
> ```bash
> nhash text lorem -p 3 --output result.txt
> ```
