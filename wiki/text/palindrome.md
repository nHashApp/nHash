# ⚡ `nhash text palindrome` - Palindrome Checker

Verifies whether a string reads the same forwards and backwards.

---

> [!NOTE]
> **Subcommand Aliases**: `pal`

## 🛠️ Usage Syntax

```bash
nhash text palindrome [arguments] [options]
```

### 📥 Input Arguments

| Argument Name | Type | Description | Required |
| :--- | :--- | :--- | :--- |
| `<text>` | `string` | String to test. | ✅ Yes |

### 💡 Live Terminal Examples

**Verify palindrome word**:
```bash
nhash text palindrome "Racecar"
```


---

> [!TIP]
> All subcommands in **nHash** inherit the global `--output` (`-o`) redirection parameter to save execution results directly to a local file.
> ```bash
> nhash text palindrome "Racecar" --output result.txt
> ```
