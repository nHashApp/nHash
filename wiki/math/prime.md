# ⚡ `nhash math prime` - Number Tester

Checks primality status, perfect number flags, and Fibonacci sequence membership of a number.

---

> [!NOTE]
> **Subcommand Aliases**: `pr`, `p`

## 🛠️ Usage Syntax

```bash
nhash math prime [arguments] [options]
```

### 📥 Input Arguments

| Argument Name | Type | Description | Required |
| :--- | :--- | :--- | :--- |
| `<number>` | `string` | Positive integer number to check. | ✅ Yes |

### 💡 Live Terminal Examples

**Check if 97 is prime**:
```bash
nhash math prime 97
```


---

> [!TIP]
> All subcommands in **nHash** inherit the global `--output` (`-o`) redirection parameter to save execution results directly to a local file.
> ```bash
> nhash math prime 97 --output result.txt
> ```
