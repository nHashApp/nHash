# ⚡ `nhash math factor` - Prime Factor Solver

Resolves positive integers into their prime factor products.

---

> [!NOTE]
> **Subcommand Aliases**: `fac`, `fact`

## 🛠️ Usage Syntax

```bash
nhash math factor [arguments] [options]
```

### 📥 Input Arguments

| Argument Name | Type | Description | Required |
| :--- | :--- | :--- | :--- |
| `<number>` | `string` | Positive integer to factorize. | ✅ Yes |

### 💡 Live Terminal Examples

**Factorize 120 into prime components**:
```bash
nhash math factor 120
```


---

> [!TIP]
> All subcommands in **nHash** inherit the global `--output` (`-o`) redirection parameter to save execution results directly to a local file.
> ```bash
> nhash math factor 120 --output result.txt
> ```
