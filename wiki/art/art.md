# 🎨 Art & Visual Utilities

Terminal graphic and text art generators using rich styled formatting and TrueColor colors.

---

## 🧩 Subcommands List

| Command | Title | Purpose |
| :--- | :--- | :--- |
| [`ascii`](./ascii.md) | Ascii Art Banner Generator | Generates large, highly-stylized ASCII text banners using built-in or custom FIGlet fonts. |
| [`box`](./box.md) | Text Box Border Drawer | Draws beautiful Spectre Panel boxes around input strings with customizable border styles. |
| [`table`](./table.md) | CSV to Terminal Grid Renderer | Translates raw comma-separated values (CSV) into truecolor styled terminal grids and tables. |
| [`gradient`](./gradient.md) | Horizontal Gradient Text Renderer | Displays strings shaded with custom horizontal hex-color transitions in the terminal. |

---

## 🚀 How to Execute

To call any feature under this module, prepend the category name `art` (or one of its aliases):

```bash
nhash art [subcommand] [arguments] [options]
```
