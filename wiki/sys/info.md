# ⚡ `nhash sys info` - Telemetry Summary

Outputs a beautiful Spectre panel showing CPU Architecture, cores count, RAM, OS, and Runtime version.

---

> [!NOTE]
> **Subcommand Aliases**: `i`

## 🛠️ Usage Syntax

```bash
nhash sys info [arguments] [options]
```

### 💡 Live Terminal Examples

**Retrieve system hardware summary**:
```bash
nhash sys info
```


---

> [!TIP]
> All subcommands in **nHash** inherit the global `--output` (`-o`) redirection parameter to save execution results directly to a local file.
> ```bash
> nhash sys info --output result.txt
> ```
